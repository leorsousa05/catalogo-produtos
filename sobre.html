L'Index

A L'Index nasceu da crença de que os objetos que escolhemos para nos cercar devem ter significado. Não se trata apenas de funcionalidade, mas de história, de proporção, de materialidade.

Cada peça deste catálogo foi selecionada como se estivesse sendo escolhida para uma galeria particular. Buscamos formas atemporais, acabamentos honestos e aquele detalhe imperceptível que separa o comum do extraordinário.

"O luxo não está no excesso, mas na precisão de cada escolha."

Este é um projeto acadêmico desenvolvido para demonstrar a construção de interfaces web modernas utilizando apenas tecnologias front-end puras. A experiência integra consumo de API, manipulação de DOM e persistência de dados no navegador.
